# 텍스트
#을 하나 쓰면 HTML의 <h1> 태그를, #을 두개 쓰면 <h2>태그를 의미한다. 즉, #은 하나에서 여섯개까지 쓸 수 있고, #이 늘어날때마다 제목의 수준은 내려간다.(보통 글씨 크기가 작아진다.)
번호 없는 리스트
-/+/* 텍스트
번호 있는 리스트
숫자. 리스트
기울인 글씨 (html의 <em>태그)
*텍스트* or _텍스트_
굵은 글씨 (html의 <strong>태그)
**텍스트** or __텍스트__
인용
> 텍스트
인용문안에 인용문을 넣으려면 >를 더 추가해주면 된다.
